== Description ==

### Translate WordPress Content

WP Translate is an easy to use plugin that will dynamically translate your WordPress content using the power of Google Transalte. Your visitors will be able read your content in their own language by simply selecting it from the available options.

This **Google Translate WordPress Plugin** dynamically adds a Google Translate widget to the upper right of your users' screen. This will allow visitors to translate your website content to any language and make your WordPress site or blog multilingual. Bring your WordPress website to the global market place today!

== Installation ==

1. Upload the `wp-translate` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
